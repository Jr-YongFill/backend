<!-- GETTING STARTED -->
## Getting Started
1. Clone the repo
    ```sh
    git clone https://github.com/Jr-YongFill/backend
    ```
2. Build Project
    ```sh
    cd ./backend
    mvn -B package -DskipTests=true --file pom.xml
    ```
3. Run
    ```sh
    java -jar ./target/server-0.0.1-SNAPSHOT.jar
    ```
